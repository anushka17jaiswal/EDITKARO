# EDITKARO
🎬 Editkaro – Editorial Portfolio Website

Overview

Editkaro is a modern, editorial-inspired portfolio website designed for a creative video editing and social media marketing agency. Inspired by the clean aesthetics of luxury magazines, the website showcases video projects through immersive storytelling, elegant typography, and cinematic layouts.

Built entirely with HTML, CSS, and JavaScript, the project focuses on delivering a premium user experience while maintaining responsiveness and smooth interactions across devices.

---

✨ Features

- 🎥 Full-screen cinematic hero section with autoplay background video
- 📰 Editorial magazine-inspired UI and typography
- 📖 Featured Stories section showcasing creative video projects
- 🎮 Portfolio categories including:
  - Football Editing
  - Anime Editing
  - Gaming Videos
  - E-Commerce Ads
  - Documentary Style
  - Color Grading
  - Short Form Videos
  - Long Form Videos
- 🎨 Smooth scroll animations using JavaScript
- 📱 Fully responsive design for desktop, tablet, and mobile
- ⚡ Lightweight and fast with no external frameworks

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Cormorant Garamond, Space Grotesk, Inter)
